# Операторы ветвления и логические операции

Решения заданий лежат в отдельных папках:

- `task1/main.cpp` — задание 1, таблица истинности.
- `task2/` — место для задания 2.
- `task3/` — место для задания 3.
- `task4/` — место для задания 4.

Чтобы запустить конкретное задание в VS Code, открой нужный `main.cpp` и нажми `Ctrl + Shift + B`.
